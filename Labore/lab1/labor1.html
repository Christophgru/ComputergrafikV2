Rendern von Grafiken mit <canvas>-2D

Diese Website erklärt die wesentlichen Bausteine zum Rendern von Grafiken innerhalb
einer Website mit der Bibliothek <canvas>-2D

<canvas> HTML Element

Die Basis für das Rendern von Grafiken
in HTML ist ein <canvas>-Element. Auf
dieses Canvas-ELement wird über die
DOM-Schnittstelle per JavaScript-Objekt
zugegriffen. JavaScript erzeugt
anschliessend einen 2D-Zeichenkontext
